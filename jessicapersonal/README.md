# Jéssica Personal - Landing Page

Landing page moderna e responsiva para Personal Trainer Jéssica, desenvolvida com HTML, CSS e JavaScript puro.

## 🎯 Objetivo

Conversão de visitantes para contato via WhatsApp, apresentando os serviços profissionais de treinamento personalizado.

## 🎨 Design

- **Cores**: Preto (#000000) e Vermelho (#E50914)
- **Estilo**: Moderno, minimalista e feminino
- **Tipografia**: Poppins (Google Fonts)
- **Layout**: Mobile-first com design responsivo

## 📱 Funcionalidades

### Seções
- **Hero**: Apresentação principal com CTA para WhatsApp
- **Sobre**: Informações profissionais e formação
- **Serviços**: Cards com serviços oferecidos
- **Diferencial**: Vantagens competitivas
- **Hipopressivo**: Técnica especializada e benefícios
- **CTA Final**: Chamada para ação principal
- **Footer**: Informações de contato

### Recursos Técnicos
- ✅ Design responsivo (mobile-first)
- ✅ Animações suaves ao scroll
- ✅ Botão flutuante do WhatsApp
- ✅ Scroll suave entre seções
- ✅ Efeitos hover interativos
- ✅ Otimização de performance
- ✅ Código semântico e acessível

## 🚀 Tecnologias

- **HTML5**: Estrutura semântica
- **CSS3**: Design moderno com animações
- **JavaScript Vanilla**: Interatividade e performance
- **Google Fonts**: Tipografia Poppins
